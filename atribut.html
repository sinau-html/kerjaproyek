Atribut HTML

- Semua elemen HTML dapat memiliki atribut
- Atribut memberikan informasi tambahan tentang elemen
- Atribut selalu ditentukan pada tag pembuka
- Atribut biasanya datang dalam pasangan nama / nilai seperti: nama = "nilai"

Atribut lang

Bahasa suatu dokumen dapat dinyatakan dalam tag <html>.
Bahasa dinyatakan dengan atribut lang.
Mendeklarasikan bahasa adalah penting untuk aplikasi aksesibilitas (pembaca layar) dan mesin pencari:

< !DOCTYPE html >
<html lang="en-US">
<body >

...

</body

Dua huruf pertama menentukan bahasa yang digunakan yaitu bahasa inggris (en). Jika terdapat dialek, gunakan dua huruf lebih, contohnya Amerika Serikat (US).

Atribut href

Link HTML didefinisikan dengan tag <a>. Alamat link yang ditentukan dalam atribut href:

<a href ="http://www.google.com"> Ini adalah link</a>
Klik untuk mencoba!

Atribut Size (ukuran gambar)

Gambar pada HTML didefinisikan dengan tag <img>. Nama file dari sumber (src), dan ukuran gambar (lebar dan tinggi) semua disediakan sebagai atribut:

<img src="img/smk.png" width="250" height="250">
Klik untuk mencoba!

Ukuran gambar ditentukan dalam pixel: width = "250" berarti lebar gambar adalah 250 pixel.

Atribut Alt

Atribut alt menetapkan teks alternatif yang akan digunakan, jika gambar tidak dapat ditampilkan.

<img src ="img/smk.png" alt="ini adalah logo smk 1 kedungwuni" width="250" height="250">
Klik untuk mencoba!
