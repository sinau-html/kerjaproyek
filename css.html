Membuat HTML menarik dengan CSS

CSS adalah singkatan dari Cascading Style Sheets. CSS menjelaskan bagaimana elemen HTML yang akan ditampilkan pada layar, kertas, atau media lainnya. CSS menghemat banyak pekerjaan. CSS dapat mengontrol tata letak beberapa halaman web sekaligus. CSS dapat ditambahkan ke elemen HTML dalam 3 cara:
1. Inline - dengan menggunakan atribut style dalam elemen HTML
2. Internal - dengan menggunakan elemen <style> pada bagian <head>
3. External - dengan menggunakan file CSS eksternal
Cara yang paling umum untuk menambahkan CSS, adalah untuk menyimpan style dalam file CSS yang terpisah. Namun, di sini kita akan menggunakan cara inline dan internal, karena ini lebih mudah untuk ditunjukkan, dan lebih mudah bagi Anda untuk mencoba sendiri.

Inline CSS

Sebuah inline CSS digunakan untuk menerapkan gaya unik untuk 1 elemen HTML. Sebuah inline CSS menggunakan atribut gaya elemen HTML. Contoh ini menetapkan warna teks elemen <h1> menjadi biru:

<h1 style="color:blue;">Ini adalah heading</h1>
Klik untuk mencoba!

Internal CSS

Sebuah CSS internal digunakan untuk menentukan gaya untuk halaman HTML. Sebuah CSS internal didefinisikan dalam <head> halaman HTML, dalam elemen <style>:

<!DOCTYPE html>
<html>
<head>
<style>
body {background-color: powderblue;}
h1   {color: blue;}
p    {color: red;}
</style>
</head>
<body>

<h1>Ini adalah Heading</h1>
<p>Ini adalah Paragraf.</p>

</body>
</html>
Klik untuk mencoba!

External CSS

CSS eksternal digunakan untuk menentukan gaya untuk banyak halaman HTML. Dengan CSS eksternal, Anda dapat mengubah tampilan seluruh situs web, HANYA dengan mengubah satu file! Untuk menggunakan CSS eksternal, tambahkan link ke dalam bagian <head> dari halaman HTML:

<!DOCTYPE html>
<html>
<head>
  <link rel="stylesheet" href="assets/css/styles.css">
</head>
<body>

<h1>Ini adalah Heading</h1>
<p>Ini adalah Paragraf.</p>

</body>
</html>
Klik untuk mencoba!

CSS eksternal dapat ditulis dalam teks editor apapun. file tidak harus berisi kode HTML, dan harus disimpan dengan ekstensi .css. Berikut adalah isi dari "styles.css" :

body {
    background-color : powderblue;
}
h1 {
    color: blue ;
}
p {
    color: red;
 }

Font CSS

Properti color pada CSS mendefinisikan warna teks yang akan digunakan. Properti CSS font-family mendefinisikan font yang akan digunakan. Properti CSS font-size mendefinisikan ukuran teks yang akan digunakan.

<!DOCTYPE html>
<html>
<head>
<style>
h1 {
    color: blue;
    font-family: verdana;
    font-size: 300%;
}
p  {
    color: red;
    font-family: courier;
    font-size: 160%;
}
</style>
</head>
<body>

<h1>Ini adalah Heading</h1>
<p>Ini adalah paragraf.</p>

</body>
</html>
Klik untuk mencoba!

Border CSS

Properti border pada CSS mendefinisikan batas pada sekitar elemen HTML:

p {
    border : 1px solid powderblue;
}
Klik untuk mencoba!

Padding CSS

Properti CSS padding mendefinisikan padding (spasi) antara teks dan border:

p {
    border : 1px solid powderblue;
    padding : 30px;
}
Klik untuk mencoba!

Margin CSS

Properti CSS margin mendefinisikan margin (ruang) di luar border:

p {
    border : 1px solid powderblue;
    margin : 50px;
}
Klik untuk mencoba!

Atribut Id

Untuk menentukan gaya tertentu untuk satu elemen khusus, tambahkan atribut id ke elemen:

<p id="p01">Aku berbeda</p>

Kemudian tentukan gaya untuk elemen dengan id tersebut:

#p01 {
    color : blue;
}
Klik untuk mencoba!

Catatan: id dari elemen harus unik dalam sebuah halaman, sehingga pemilih id digunakan untuk memilih salah satu elemen yang unik!

Atribut Class

Untuk menentukan gaya untuk jenis khusus dari elemen, tambahkan atribut class untuk elemen:

<p class="error">Aku berbeda</p>

kemudian tentukan gaya untuk elemen dengan class khusus:

p.error {
    color: red;
}
Klik untuk mencoba!

Referensi External

CSS external bisa direferensikan dengan full URL atau dengan path yang sesuai dengan halaman web tersebut. Ini adalah contoh penggunaan full URL :

<link rel="stylesheet" href="http://www.w3schools.com/html/styles.css">

Contoh ini link ke CSS yang terletak di folder html di situs web saat ini:

<link rel="stylesheet" href="/html/styles.css">

Contoh ini link ke CSS yang terletak di folder yang sama dengan halaman web saat ini:

<link rel="stylesheet" href="styles.css">
