Elemen HTML

Elemen pada HTML sering tedapat Tag Pembuka dan Tag Penutup, dengan konten yang diselipkan di dalam kedua tag tersebut. Contohnya :
<tag pembuka> Isi konten </ tag penutup>

Elemen HTML adalah semua yang terdapat pada tag pembuka sampai tag penutup.

Elemen HTML yang tidak terdapat konten di dalamnya disebut elemen kosong (empty elements). Elemen kosong tidak memiliki tag penutup, seperti elemen <br> (yang berguna untuk mengganti baris).


HTML Bersarang

Penjelasan

Tag <html> mendefinisikan seluruh dokumen html. Di dalamnya terdapat tag pembuka berupa <html> dan tag penutup berupa </html>. Tag <body> mendefinisikan tubuh dari dokumen html. Di dalamnya terdapat tag pembuka berupa <body> dan tag penutup </body>. Di dalam contoh di atas juga terdapat 2 elemen html lainnya yaitu <h1> dan <p>. Penjelasan dari dua elemen html tersebut ada di bawah ini.

<h1>Ini adalah Heading </h1>

Tag <h1> mendefinisikan heading. Di dalamnya terdapat tag pembuka berupa <h1> dan tag penutup berupa </h1> . Konten dari elemen tersebut adalah "Ini adalah Heading"

<p>Ini adalah Paragraf </p>

Tag <h1> mendefinisikan paragraf. Di dalamnya terdapat tag pembuka berupa <p> dan tag penutup berupa </p> . Konten dari elemen tersebut adalah "Ini adalah Paragraf"

Elemen HTML Kosong

Elemen HTML tanpa konten disebut dengan Elemen Kosong. Contohnya adalah tag <br> yang tidak membutuhkan tag penutup. Elemen kosong bisa "ditutup" pada tag pembuka, contohnya : <br />. HTML5 tidak mewajibkan elemen kosong untuk diberi tag penutup. Tapi jika anda ingin validasi yang ketat dan ingin dokumen anda bisa dibaca oleh XML parsers, anda harus menutup semua elemen HTML pada dokumen anda.



Tulis Tag dengan Huruf Kecil

Tag HTML tidak case sensitive, artinya tidak membedakan penggunaan huruf besar dan kecil. <P> (dengan huruf besar) sama artinya dengan <p>.
Standar HTML5 tidak mewajibkan penggunaan huruf kecil, namun W3C MEREKOMENDASIKAN huruf kecil pada HTML, dan MEMBUTUHKAN huruf kecil pada tipe dokumen yang lebih ketat seperti XHTML.
