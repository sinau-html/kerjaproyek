File path (letak file) Pada HTML


LokasiDeskripsi
<img src="picture.jpg"> picture.jpg terletak di folder yang sama dengan halaman saat ini
<img src="images/picture.jpg"> picture.jpg terletak di dalam folder image yang terletak di folder saat ini
<img src="/images/picture.jpg"> picture.jpg terletak di folder image yang terletak pada root web saat ini
<img src="../picture.jpg"> picture.jpg terletak di folder satu tingkat dari folder saat ini

File path (lokasi file) pada HTML

Sebuah path file menjelaskan lokasi file dalam struktur folder situs Web. path file digunakan saat menghubungkan ke file eksternal seperti:

  • Gambar
  • Halaman web
  • Style sheet
  • Javascript

Absolute file path

Absolut file path adalah URL lengkap yang menuju ke file internet

Contoh

<img src="https://www.w3schools.com/images/picture.jpg" alt="Mountain">
Klik untuk mencoba!

Relative path file

Sebuah Relative path file mengarahkan ke file pada halaman saat ini. Pada contoh kali ini file path mengarahkan file didalam folder image yang terletak pada root web saat ini

Contoh

<img src="images/picture.jpg" alt="Mountain">
