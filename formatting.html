Format Elemen HTML

Format elemen dirancang untuk menampilkan jenis khusus dari teks:

  • <b> - Teks Tebal
  • <strong> - Teks Penting
  • <i> - Teks Miring
  • <em> - Teks Emphasized
  • <mark> - Teks bertanda
  • <small> - Teks Kecil
  • <del> - Teks Dihapus
  • <ins> - Teks Ditambahkan
  • <sub> - Teks Subscript
  • <sup> - Teks Superscript

Elemen HTML <b> dan <strong>

Tag <b> mendefinisikan teks tebal, tanpa tambahan penting. HTML <strong> mendefinisikan teks yang kuat, dengan menambahkan semantik penting berupa "strong".

<b>Teks Tebal</b>
<strong>Teks Strong</strong>
Klik untuk mencoba!

Elemen HTML <i> dan <em>

Tag <i> mendefinisikan teks miring, tanpa tambahan penting. HTML <em> mendefinisikan teks yang emphasized (diberi tekanan) dengan menambahkan semantik penting.

<i>Teks ini miring</i>
<em>Teks ini emphasized (diberi tekanan)</em>
Klik untuk mencoba!

Catatan : browser menampilkan <strong> sama seperti <b> dan menampilkan <em> sama seperti <i>. Tapi ada perbedaan diantara keduanya. tag <b> dan <i> menetapkan teks tebal dan miring, tapi tag <strong> dan <em> menandakan bahwa teks tersebut PENTING.

Elemen HTML <del>

Tag <del> menetapkan teks yang dibusek dihapus.

Elemen HTML <ins>

Tag <ins> menetapkan teks yang ditambahkan.

Elemen HTML <sub>

Tag <sub> menetapkan teks subscript atau berada di bawah garis.

Elemen HTML <sup>

Tag <sup> menetapkan teks yang ditulis di atas.
