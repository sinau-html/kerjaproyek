Syntax Gambar

Dalam HTML, gambar didefinisikan dengan tag <img>. Tag <img> kosong, hanya berisi atribut saja, dan tidak memiliki tag penutup. Atribut src menentukan URL (alamat web) gambar:

<img src="url" alt="some_text" style="width:width;height:height;">

Atribut Alt

Atribut alt menyediakan teks alternatif untuk gambar, jika pengguna karena beberapa alasan tidak dapat melihatnya (karena koneksi lambat, kesalahan pada atribut src, atau jika pengguna menggunakan pembaca layar). Jika browser tidak dapat menemukan gambar, maka akan muncul nilai atribut alt:

<img src="wrongname.gif" alt="HTML5 Icon" style="width:128px;height:128px;">
Klik untuk mencoba!

Ukuran Gambar - Lebar(Width) dan Tinggi(Height)

Anda dapat menggunakan atribut style untuk menentukan lebar dan tinggi gambar. Nilai ditentukan dalam piksel (gunakan px setelah nilainya):

<img src="html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">
Klik untuk mencoba!

Sebagai alternatif, Anda bisa menggunakan atribut width dan height. Di sini, nilai ditentukan dalam piksel secara default:

<img src="html5.gif" alt="HTML5 Icon" width="128" height="128">
Klik untuk mencoba!

Catatan: Selalu tentukan lebar dan tinggi gambar. Jika lebar dan tinggi tidak ditentukan, halaman akan berkedip saat gambar dimuat.

Width dan Height atau Style?

Atribut width, height, dan style berlaku di HTML5. Namun, sebaiknya gunakan atribut style. Ini mencegah lembaran gaya internal atau eksternal mengubah ukuran gambar aslinya:

<!DOCTYPE html>
<html>
<head>
<style>
img {
    width:100%;
}
</style>
</head>
<body>

<img src="html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">
<img src="html5.gif" alt="HTML5 Icon" width="128" height="128">

</body>
</html>
