Heading HTML

Heading pada HTML didefinisikan dengan tag <h1> sampai <h6>. h1 mendefinisikan heading yang paling penting, dan h6 mendefinisikan heading yang kurang penting.


Klik untuk mencoba!

Catatan : Browser secara otomatis akan memberikan jarak (margin) sebelum dan sesudah Heading.

Heading itu Penting

Mesin Pencari (search engine) menggunakan heading untuk mengindex struktur dan konten pada website anda. Menggunakan heading itu penting untuk menunjukkan struktur dokumen. <h1> digunakan untuk heading yang paling utama, diikuti oleh <h2>, lalu <h3> dan seterusnya.

Catatan : Gunakan Heading HTML hanya untuk Heading, BUKAN untuk membuat teks menjadi Besar atau Tebal.

HTML Horizontal Rules

Tag <hr> mendefinisikan jeda tematik di halaman HTML, dan yang paling sering ditampilkan sebagai garis horisontal. Elemen <hr> digunakan untuk memisahkan konten (atau menentukan perubahan) dalam sebuah halaman HTML:

<h1>Ini adalah heading 1 </h1>
<p>Ini adalah teks. </p>
<hr>
<h2>Ini adalah heading 2</h2 >
<p> Ini teks yang lainnya. </p>
<hr>
Klik untuk mencoba!

Elemen HTML <head>

Elemen HTML <head> tidak ada hubungannya dengan heading pada HTML. Elemen <head> merupakan wadah untuk metadata. metadata HTML adalah data tentang dokumen HTML. Metadata tidak ditampilkan pada halaman website. Elemen <head> ditempatkan di antara tag <html> dan tag <body>:

<!DOCTYPE html>
<html>

<head>
  <title> My First HTML</title >
  <meta charset="UTF-8" >
</head>

<body>
.
.
.

Catatan : Metadata biasanya menentukan judul dokumen, set karakter, gaya, link, script, dan informasi meta lainnya.
