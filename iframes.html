Iframes Pada HTML

Sebuah Iframe pada HTML digunakan untuk menamilkan sebuah situs web di dalam situs web

Sintaks iframes

Pada HTML iframes didefinisikan dengan tag <iframes>

<iframe src="URL"></iframe>

Atribut src menentukan URL (alamat web) yang akan ditampilkan pada halaman frame inline.

Iframe - Mengatur panjang dan lebar

Gunakan Atrribut HEIGHT dan LENGTH untuk menuntukan ukuran Iframe. secara default Attribute ini menggunakan ukuran pixel

<iframe src="demo_iframe.htm" height="200" width="300"></iframe>
Klik untuk mencoba!

Iframe - Menghilangkan batas garis (border)

Secara default sebuah iframe memiliki garis batas di sekitarnya. Untuk menghilangkanya gunakan Attribut STYLE dan css BORDER

<iframe src="demo_iframe.htm" style="border:none;"></iframe>
Klik untuk mencoba!
