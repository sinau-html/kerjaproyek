Komentar HTML

Tag komentar digunakan untuk menambahkan komentar pada dokumen HTML. Anda bisa membuat komentar dengan sintaks berikut ini :

<!-- Tulis komentar anda disini -->

Perhatikan bahwa pada komentar terdapat tanda seru (!) pada bagian tag pembuka, tp tidak terdapat tanda tersebut pada tag penutup

Catatan: Komentar tidak ditampilkan pada browser, tetapi bisa membantu pada dokumen HTML yang anda buat.

Dengan komentar anda bisa memberi informasi tambahan pada dokumen HTML yang anda buat.

<!-- Ini komentar -->

<p>Ini paragraf.</p>

<!-- Anda bisa memberi informasi tambahan disini -->
Klik untuk mencoba!

Komentar bersyarat

Anda mungkin akan menemui komentar bersyarat di HTML seperti di bawah ini:

<!--[if IE 9]>
    .... Tulis kode HTML disini ....
<![endif]-->

Komentar bersyarat mendefinisikan beberapa tag HTML yang harus dijalankan oleh Internet Explorer saja.
