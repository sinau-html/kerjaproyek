Link HTML

Link ditemukan di hampir semua halaman web. Link memungkinkan pengguna untuk mengunjungi dari halaman ke halaman lainnya.

Link HTML - Hyperlink

Link HTML adalah hyperlink. Anda dapat mengklik link dan melompat ke dokumen lain. Bila Anda memindahkan mouse di atas link, panah mouse akan berubah menjadi tangan kecil.

Catatan: Sebuah link tidak harus berupa teks. Hal ini dapat gambar atau elemen lain HTML.

Link HTML - Sintaks

Pada HTML, link didefinisikan dengan tag <a> :

<a href="url">Teks Link</a>
<a href="http://www.google.com/html/">Kunjungi Google.com !</a>
Klik untuk mencoba!

Atribut href men-spesifikasikan alamat tujuan dari link (www.google.com).
link teks menetapkan bagian yang terlihat pada browser (Kunjungi Google.com !). Mengklik pada tulisan link akan membawa anda ke halaman web tujuan.

Catatan: Tanpa garis miring pada alamat subfolder, Anda mungkin akan menghasilkan dua permintaan ke server. Banyak server secara otomatis akan menambahkan garis miring ke alamat, dan kemudian membuat permintaan baru.

Link Lokal

Contoh di atas menggunakan URL absolut (sebuah alamat web penuh). Sebuah link lokal (link ke situs web yang sama) ditentukan dengan URL relatif (tanpa http: // www ....).

<a href="html_images.asp">Gambar HTML</a>

Link HTML Warna

Secara default, link akan muncul seperti ini (di semua browser):
1. Link yang belum dikunjungi adalah digarisbawahi dan berwarna biru
2. Link yang sedang dikunjungi adalah digarisbawahi dan berwarna ungu
3. Link yang aktif akan digarisbawahi dan berwarna merah
Anda bisa mengubah warna default dengan contoh dibawah ini :

<style>
a:link    {color:green; background-color:transparent; text-decoration:none}
a:visited {color:pink; background-color:transparent; text-decoration:none}
a:hover   {color:red; background-color:transparent; text-decoration:underline}
a:active  {color:yellow; background-color:transparent; text-decoration:underline}
</style>
Klik untuk mencoba!

Link HTML - Atribut Target

Atribut target menentukan di mana untuk membuka link dokumen. Atribut target dapat memiliki salah satu dari nilai berikut:
1. _blank : Membuka link dokumen pada tab baru.
2. _self : Membuka link dokumen di jendela / tab yang sama seperti yang diklik (ini adalah default)
3. _parent : Membuka dokumen terkait dalam kerangka induk
4. _top : Membuka dokumen terkait dalam tubuh penuh jendela
5.framename : Membuka dokumen terkait dalam bingkai bernama
Contoh di bawah ini akan membuka link dokumen pada tab baru di browser.

<a href="http://www.google.com/" target="_blank">Kunjungi situs google!</a>
Klik untuk mencoba!

Link HTML - Gambar sebagai Link

Contoh di bawah ini akan menunjukkan gambar sebagai link.

<a href="http://www.smk1kedungwuni.sch.id">
  <img src="img/smk.png" alt="Logo SMK" style="width:42px;height:42px;border:0;">
</a>
Klik untuk mencoba!

Catatan: border: 0; ditambahkan untuk mencegah IE9 (dan sebelumnya) dari menampilkan border di sekitar gambar (ketika gambar menjadi sebuah link).


Membuat Sebuah Bookmark

Bookmark HTML digunakan untuk memungkinkan pembaca untuk melompat ke bagian tertentu dari halaman Web. Bookmark dapat berguna jika halaman Web Anda sangat panjang. Untuk membuat bookmark, Anda harus terlebih dahulu membuat bookmark, dan kemudian menambahkan link untuk itu. Ketika link diklik, halaman akan bergulir ke lokasi yang telah diberi bookmark atau penunjuk tersebut.

Contoh

Pertama, buatlah bookmark dengan atribut id:

<h2 id="penting">Bagian Penting </h2>

Kemudian, tambahkan link ke bookmark ("Bagian Penting"), dari dalam halaman yang sama:

<a href="#penting">Lihat Bagian Penting!</a>

Atau, tambahkan link ke bookmark ("Bagian Penting"), dari halaman lain:

<a href="html_tips.html#penting">Lihat Bagian Penting!</a>
Klik untuk mencoba!

External Path

Halaman eksternal dapat dirujuk dengan URL penuh atau dengan path relatif ke halaman web saat ini. Contoh ini menggunakan URL lengkap untuk link ke halaman web:

<a href="http://www.w3schools.com/html/default.asp">HTML tutorial</a>
Klik untuk mencoba!

Contoh ini link ke halaman yang terletak di folder html di situs web saat ini:

<a href="/html/default.asp">HTML tutorial</a>

Contoh ini link ke halaman yang terletak di folder yang sama dengan halaman saat ini:

<a href="default.asp">HTML tutorial</a>
