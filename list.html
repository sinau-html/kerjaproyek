    Unordered List Pada HTML - Memilih Ikon Penanda List

    Untuk memilih ikon penanda list gunakan css "list-style-type"

    attribut Descripsi
    disc Mengatur penanda list menjadi titik hitam (default)
    circle Mengatur penanda list menjadi lingkaran
    square Mengatur penanda list menjadi persegi
    none Mengatur penanda list menjadi kosong

    contoh - Disc

    <ul style="list-style-type:disc">
      <li>Coffee</li>
      <li>Tea</li>
      <li>Milk</li>
    </ul>
    Klik untuk mencoba!

    Contoh - circle

    <ul style="list-style-type:circle">

    Ordered List Pada HTML

    Apabila ingin menambah daftar pada HTML dengan nomor urut, gunakan "ordered list". ordered list pada html maka menggunakan tag "<ol>"

    contoh

    <ol>
      <li>Kopi</li>
      <li>Teh</li>
      <li>Susu</li>
    </ol>

    Klik untuk mencoba!

    Ordered List Pada HTML - Attribut

    Perhatikan tipe ikon penanda list pada ordered list berikut

    Tipe Deskripsi
    type="1" Penanda pada list akan menggunakan angka (default)
    type="A" Penanda pada list akan menggunakan huruf besar
    type="a" Penanda pada list akan menggunakan huruf kecil
    type="I" Penanda pada list akan menggunakan angka romawi besar
    type="i" Penanda pada list akan menggunakan angka romawi kecil

    Angka:

    <ol type="1">
      <li>Kopi</li>
      <li>Teh</li>
      <li>Susu</li>
    </ol>
    Klik untuk mencoba!

    Huruf besar:

    <ol type="A">

      <li>Teh</li>
      <li>Susu</li>
    </ol>
    Klik untuk mencoba!

    </ol>

    Description List Pada HTML

    Description list adalah daftar yang mempunyai deskripsi) gunakan tag <dl> untuk memulai description list,gunakan <dt> untuk mendefinisikan list, gunakan <dd>untuk memberi penjelasan pada list

    Contoh

    <dl>
      <dt>Kopi</dt>
      <dd>- Minuman dari biji kopi</dd>
      <dt>Teh</dt>
      <dd>- minuman dari daun teh</dd>
    </dl>
    Klik untuk mencoba!

    Nested List (List Bersarang) Pada HTML

    List Bersarang adalah list yang berada dalam list

    Contoh

    <ul>
      <li>Kopi</li>
      <li>Teh
        <ul>
          <li>Teh hitam</li>
          <li>Teh hijau</li>
        </ul>
      </li>
      <li>Susu</li>
    </ul>
    Klik untuk mencoba!
