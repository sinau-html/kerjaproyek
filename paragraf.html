Paragraf HTML

Elemen HTML <p> mendefinisikan suatu paragraf.

<p>Ini adalah paragraf.</p >
<p>Ini juga paragraf.</p >
Klik untuk mencoba!

Catatan : Browser secara otomatis akan memberikan jarak (margin) sebelum dan sesudah Paragraf.

Tampilan HTML

Anda tidak bisa yakin bagaimana HTML akan ditampilkan. Besar atau kecil layar, dan mengubah ukuran jendela akan menciptakan hasil yang berbeda. Dengan HTML, Anda tidak dapat mengubah output dengan menambahkan spasi ekstra atau baris ekstra dalam kode HTML Anda. Browser akan menghapus spasi ekstra dan garis ekstra saat halaman ditampilkan:

<p>
This paragraph
contains a lot of lines
in the source code,
but the browser
ignores it.
</p >

Jangan lupakan tag penutup

Beberapa elemen HTML akan tetap muncul dengan baik meskipun anda lupa menulis tag penutupnya. Contoh di bawah ini akan berjalan di semua browser, karena tag penutup bersifat opsional.

<p>Ini adalah paragraf.
<p>Ini juga paragraf.
Klik untuk mencoba!

Jangan pernah lakukan hal ini! Karena akan membuat hasil koding tidak sesuai keinginan dan dapat menimbulkan error!

HTML Ganti Baris

Elemen HTML <br> mendefinisikan ganti baris. Gunakan <br> untuk melakukan pergantian baris tanpa harus membuat paragraf baru. Tag <br> adalah tag kosong, yang berarti tag ini tidak membutuhkan tag penutup.

<p>Ini adalah<br>sebuah paragraf <br>dengan pergantian baris.</p>
Klik untuk mencoba!

Elemen HTML <pre>

Elemen HTML <pre> mendefinisikan teks terformat. Teks di dalam elemen <pre> mempertahankan jarak dan pergantian baris.

<pre>
  My Bonnie lies over the ocean.

  My Bonnie lies over the sea.

  My Bonnie lies over the ocean.

  Oh, bring back my Bonnie to me.
</pre>
Klik untuk mencoba!
