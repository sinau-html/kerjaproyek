HTML Style Atribut


Pengaturan gaya elemen HTML, dapat dilakukan dengan atribut style. Atribut style HTML memiliki sintaks berikut:

<tagname style="property:value;">

Property adalah properti dari CSS, dan value adalah nilai dari properti tersebut.

Warna Background HTML

Properti background-color menetapkan warna background (latar belakang) dari suatu elemen HTML. Contoh di bawah ini adalah merubah warna background suatu elemen HTML menjadi powderblue.

<body style="background-color:powderblue;">

<h1>Ini adalah Heading</h1>
<p>Ini adalah Paragraf.</p>

</body>

Warna Teks HTML

Properti color menetapkan warna dari suatu teks

<h1 style="color:blue;">Ini adalah Heading</h1>
<p style="color:red;">Ini adalah Paragraf.</p>
Klik untuk mencoba!

Font HTML

Properti font-family menentukan jenis font yang akan digunakan pada suatu elemen HTML.

<h1 style="font-family:verdana;">Ini adalah Heading</h1>
<p style="font-family:courier;">Ini adalah Paragraf.</p>
Klik untuk mencoba!

Ukuran Teks HTML

Properti font-size menetapkan ukuran font dalam suatu elemen HTML.

<h1 style="font-size:300%;">Ini adalah Heading</h1>
<p style="font-size:160%;">Ini adalah Paragraf.</p>
Klik untuk mencoba!

Alinea Teks HTML

Properti text-align menetapkan alinea dari suatu elemen HTML.

<h1 style="text-align:center;">Heading berada di tengah</h1>
<p style="text-align:center;">Paragraf berada di tengah.</p>
