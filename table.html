Tabel HTML


Definisi sebuah tabel pada HTML

Sebuah tabel pada HTML di definisikan menggunakan tag <table>

Setiap baris tabel didefinisikan dengan tag . Sebuah header tabel didefinisikan dengan tag . Secara default, judul tabel tebal dan berpusat. Sebuah data / sel tabel didefinisikan dengan tag .

Contoh:

<table style="width:100%">
  <tr>
    <th>Nama Depan</th>
    <th>Nama Belakang</th>
    <th>Umur</th>
  </tr>
  <tr>
    <td>Agus</td>
    <td>Nia</td>
    <td>50</td>
  </tr>
  <tr>
    <td>Sebastian</td>
    <td>Yuliana</td>
    <td>94</td>
  </tr>
</table>
Klik untuk mencoba!

HTML Tabel - Menambahkan Border

Jika Anda tidak menentukan border pada tabel, maka akan ditampilkan tanpa border. Sebuah border diatur menggunakan properti css border

Contoh:

table, th, td {
    border: 1px solid black;
}
Klik untuk mencoba!

HTML Tabel - Sel yang menjangkau banyak kolom

Agar Sel dapat menjangkau banyak kolom gunakan tag <colspan>

Contoh:

<table style="width:100%">
  <tr>
    <th>Nama</th>
    <th colspan="2">Nomor HP</th>
  </tr>
  <tr>
    <td>Rahayu</td>
    <td>08532476457</td>
    <td>08943357389</td>
  </tr>
</table>
Klik untuk mencoba!

HTML Tabel - Kolom yang menjangkau banyak sel

Agar kolom dapat menjangkau banyak sel gunakan tag <rowspan>

Contoh:

<table style="width:100%">
  <tr>
    <th>Nama:</th>
    <td>Rahayu</td>
  </tr>
  <tr>
    <th rowspan="2">No hp</th>
    <td>08532476457<</td>
  </tr>
  <tr>
    <td>08943357389</td>
  </tr>
</table>
Klik untuk mencoba!

HTML Tabel - Menambahkan Judul

Untuk mnambahkan judul gunakan tag <caption>

Contoh:

<table style="width:100%">
  <caption>Pengeluaran bulanan</caption>
  <tr>
    <th>Bulan</th>
    <th>Pengeluaran</th>
  </tr>
  <tr>
    <td>Januari</td>
    <td>Rp.300.000,00</td>
  </tr>
  <tr>
    <td>Februaru</td>
    <td>Rp.150.000,00</td>
  </tr>
</table>
